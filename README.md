# NutriFit Pro рҹҸӢпёҸвҖҚвҷӮпёҸ

A powerful, smart personal fitness and macro tracking application originally built with **CustomTkinter** for Desktop, and upgraded to **Streamlit** for a robust, interactive Web Dashboard experience.

## Features вңЁ
- **Interactive Dashboards**: Live progress bars tracking your exact Caloric and Protein targets.
- **Weight Tracking & Trends**: Visualize your 7-day macronutrient rolling averages and weight progression via interactive Matplotlib/Altair charts.
- **Custom Food Database**: Dynamically log and search custom foods through the native JSON architecture.
- **Bulk Mode**: A one-click toggle to scale your dietary goals by +20% for rapid gains.

## Running the Web Version (Streamlit)
```bash
conda activate nutrifit
streamlit run streamlit_app.py
```

## Running the Desktop Version (CustomTkinter)
```bash
conda activate nutrifit
python add_meal.py
```
